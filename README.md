### Installation & Usage:

#### Linux :
```
wget "https://github.com/MMT007/Termux-YTD/raw/master/bin/linux-ytd"
chmod +x linux-ytd 
mv "linux-ytd" "/usr/bin/"
```

#### Termux :

IF Git Is NOT Installed:
```
apt update
apt upgrade
pkg install git -y 
```

If Git Is Installed:
```
git clone https://github.com/MMT007/Termux-YTD.git
cd Termux-YTD/etc
bash install.sh
```
